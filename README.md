# SCIM Fuzzy PID Controller

## 📌 Project Overview

This project presents the design and implementation of advanced control techniques for a **Squirrel Cage Induction Motor (SCIM)** using **PID**, **PI**, and **Fuzzy Logic Controllers** in the MATLAB/Simulink environment. The primary objective is to improve motor speed regulation, dynamic response, and system stability under varying operating conditions.

The project evaluates and compares the performance of conventional and intelligent control methods for enhancing the efficiency and accuracy of induction motor control systems.

---

## 🎯 Objectives

* Design and simulate SCIM control models using MATLAB/Simulink
* Implement PI, PID, and Fuzzy Logic Controllers
* Analyze motor speed response and controller performance
* Improve system stability and transient characteristics
* Compare controller efficiency under different conditions

## ⚙️ Controllers Implemented

### 🔹 PI Controller

Used for basic speed regulation and steady-state error reduction in the induction motor system.

### 🔹 PID Controller

Enhances transient response, reduces overshoot, and improves system stability through proportional, integral, and derivative control actions.

### 🔹 Fuzzy Logic Controller

Implements intelligent decision-making using fuzzy inference rules to achieve adaptive and precise motor speed control.
